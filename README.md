# Kumpul Bagi Parser
Direct Link Download untuk Setiap Halaman KumpulBagi. 

**Instalasi**
```sh
npm install -g kumpulbagi-parser
```

**Penggunaan**
```sh
kumpulbagi-parser --url="http://kumpulbagi.id/inaho-kaizuka/detective-conan-1-774-ongoing-49087" --output="./conan"
```
